# Painkiller Wireframes

Live interactive prototype for the Painkiller product design.

**Live:** https://painkiller-wires.vercel.app

## Getting Started

1. **Clone this repo:**
   ```
   git clone https://github.com/aromanarguello/painkiller-wires.git
   cd painkiller-wires
   ```

2. **Open locally:** Double-click `index.html` or drag it into your browser. No install needed.

3. **Make changes:** Edit `index.html` in any text editor or with Claude Code.

4. **Deploy:** Push to `main` and Vercel deploys automatically.
   ```
   git add index.html
   git commit -m "what you changed"
   git push
   ```

## Using Claude Code

Open this project folder in Claude Code. It will read the `CLAUDE.md` file automatically and understand the full structure of the wireframes — which screens exist, how the CSS is organized, and how to make changes safely.

Just tell Claude Code what you want in plain language:
- "Change the dashboard header text to say Welcome Back"
- "Add a new card to the expenses screen for insurance"
- "Make the primary buttons blue instead of black"
- "Add a new screen for notifications"

Claude Code will edit the file and you can preview by opening `index.html` in your browser. When it looks right, ask Claude Code to push it.
